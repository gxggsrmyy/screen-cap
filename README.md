# screen-cap
录屏软件，基于python3开发。

## Issues
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/hai2007/screen-cap/issues)，欢迎参与维护！

## How to use?

首先需要安装必要的依赖：

```
pip install -U wxPython basic-toolkit pillow opencv-python pyautogui
```

然后启动运行：

```
python ./src/main.py
```
